# shadcn/ui Components

This directory will contain shadcn/ui components that you add using the CLI.

To add a component, run:

```bash
npx shadcn@latest add [component-name]
```

For example:

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add dialog
```

Components will be copied here and you can freely modify them as needed.
